# 🤖 AI Brochure Generator

An AI-powered web application that generates beautiful and structured brochures from website content using **OpenAI client** and **Groq API**.

---

## 🚀 Features

- 🌐 Extracts content from any website URL  
- 🧠 Uses AI to summarize and structure information  
- 📄 Generates clean, brochure-style output  
- ⚡ Fast inference powered by Groq API  
- 🎨 Simple and interactive UI (Gradio-based)  

---

## 🛠️ Tech Stack

- **Python**
- **OpenAI Client**
- **Groq API**
- **Gradio**
- **BeautifulSoup (for web scraping)**
